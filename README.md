# AI Handball Analyse

## Project Overview
This project aims to utilize artificial intelligence techniques to analyze handball games, providing insights on player performance, team strategies, and game outcomes.

## Features
- Detailed analysis of player movements and actions
- Insights into team strategies based on historical data
- Visualization tools for understanding game dynamics
- Machine learning models to predict game outcomes based on various metrics

## Installation
To get started with the project, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/LennartDeckert/AI-Handball-Analyse.git
cd AI-Handball-Analyse
# Install dependencies
pip install -r requirements.txt
```

## Usage
After setting up the project, you can analyze games using the following commands:

```bash
python analyze_game.py --input <input_file>
```

Replace `<input_file>` with the path to your game data.
